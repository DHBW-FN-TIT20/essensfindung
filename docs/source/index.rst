Welcome to Essensfindung's documentation!
=========================================

**Essensfindung** is a webapplication to search for a random restaurant or recipe with additional filters.
The Project is written in Python and uses FastAPI and Jinja as the main Frameworks. To collect the Information of the restaurants we use the GoogleAPI.
